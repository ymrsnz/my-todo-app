# My Todo App

A simple and elegant React-based todo list application for managing your daily tasks.

## Features

- ✅ Add new todos
- ✅ Mark todos as complete/incomplete
- ✅ Delete todos
- ✅ Clean and intuitive UI
- ✅ Responsive design

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/ymrsnz/my-todo-app.git
cd my-todo-app
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

The app will open in your browser at [http://localhost:3000](http://localhost:3000).

## Usage

1. Type a todo in the input field
2. Click "Add" or press Enter to add it to your list
3. Click the checkbox to mark a todo as complete
4. Click "Delete" to remove a todo from your list

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm build` - Builds the app for production
- `npm test` - Runs the test suite

## Technologies Used

- React 18
- CSS3
- JavaScript ES6+
